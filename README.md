# Minboot for Sunxi
A bootable Stub for Sunxi.

## Requirements
- GNU make
- GCC (host and armhf)

## How To Build

From your command line:

```bash
git clone https://github.com/seorospa/minboot-sunxi.git
cd minboot-sunxi
make
```
